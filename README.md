# Node js crawler IMAGES

## Install

To install application

```npm i```

or if you have yarn

```yarn```
## Develop

To get app in dev state

```npm run serve```


## Docker
### Build

To build docker image

```docker build --tag crawler:0.0.1```

### Run

To run docker image

```docker run crawler:0.0.1```

### CLI run

```node bin/cli.js scan --domain <some domain>```
